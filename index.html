The pupose of this website is to list the Git,linux and Vi commands

I am learning to make a website using command line and host it using github pages

Linux Commands

  • CD = change directory
  • MKdir = make Directory
  • Ls = List Files

Vi Commands

  • I = insert mode
  • Escape = get out of insert mode.
  • Shiftkey + colon = command prompt for vi
  • Q = Command to quit
  • WQ = Write and Quit

Git Commands

  • Git init = create a repository
  • Git add filename = add the file to git in preparation for the commit
  • Git commit -m “message” = do the commit to make changes
  • Git checkout -b branchname = make a new branch and checkout
  • Git checkout branchname = checkout branch
  • Git merge BranchName = merge the branch Note: you have to be on the branch that you want to merge to
